# Easy Harness Trial Guide

## What Easy Harness Is For Right Now
Easy Harness records custom wire harness requests, organizes intake details, and moves each request toward review, quotation, and order confirmation.

Current public entry paths:
- AI Agent (AI Agent / AI 智能录入)
- Configurator (Configurator / 结构化配置)
- Upload Intake (Upload Intake / 上传资料录入)

## When To Use Each Path
- AI Agent (AI Agent / AI 智能录入): Use this path for open-ended requests, mixed requirements, or requests that are easier to describe in plain language.
- Configurator (Configurator / 结构化配置): Use this path for one clear structured left-to-right harness path that fits the current canvas boundary.
- Upload Intake (Upload Intake / 上传资料录入): Use this path when drawings, photos, BOMs, pinouts, sketches, or other files explain the request more clearly.

## Basic Public Flow
1. Submit a request from one of the public entry paths.
2. Open Request Status (Request Status / 请求状态) to follow progress.
3. Open Review Order (Review Order / 订单复核) or Order Draft (Order Draft / 订单草稿) when it becomes available.

## Configurator Quick Instructions
1. Add a left connector.
2. Add one or more elements if needed.
3. Add a right connector.
4. Connect the path with wires.
5. Complete the required fields for connectors, elements, and wires.
6. Select Review Order (Review Order / 复核订单) when the path is ready.

## What To Expect After Submission
- The request is recorded and receives a request ID.
- Request Status (Request Status / 请求状态) shows the current stage.
- Some structured configurator requests can move directly to Review Order (Review Order / 复核订单).
- Some requests require draft preparation before final confirmation.

## Internal Ops Use
- Ops Access (Ops Access / 内部入口): Enter the shared passcode to open internal handling.
- Request Inbox (Request Inbox / 请求收件箱): Review incoming requests.
- Request Detail (Request Detail / 请求详情): Read the intake record, prepare the draft, and update status.
- Draft preparation: Complete structured summary, assumptions, missing items, quotation notes, and manufacturing notes.
- Export Editable XLSX (Export Editable XLSX / 导出可编辑表格): Download the working handoff file for further internal editing.

## Short Testing Checklist
- Submit one AI request through AI Agent (AI Agent / AI 智能录入).
- Submit one Upload request through Upload Intake (Upload Intake / 上传资料录入).
- Submit one Configurator request through Configurator (Configurator / 结构化配置) if the structured path can be completed.
- Open Request Status (Request Status / 请求状态).
- Open Review Order (Review Order / 复核订单) or Order Draft (Order Draft / 订单草稿) when available.

## Known Prototype Boundaries
- Configurator (Configurator / 结构化配置) covers only a bounded structured path.
- Some requests still require internal preparation before final confirmation.
- Quote values may remain placeholder values during the prototype stage.
